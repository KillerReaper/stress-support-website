# Stress Management Through Social Support

## Project Overview
This project is a health promotion website created for a course assignment in **Stress Management (HEED-213-600)** at **Howard Community College**. The website is designed to educate college students on how to manage stress using social support techniques in a clear, simple, and practical way.

**Instructor:** Melanie Nystrom, MA

## Target Population
College students who experience stress from academic pressure, deadlines, and balancing responsibilities.

## Stress Management Technique
**Social Support**

Social support involves receiving help, encouragement, and understanding from others such as friends, family, classmates, or support groups.

## Purpose of the Website
The website is designed to:
- Explain what stress is and how it affects students  
- Introduce social support as a stress management strategy  
- Show why social support is effective  
- Provide real-life examples of how students can use it  

## Technology Used
- HTML  
- CSS  
- JavaScript (optional)  
- GitHub Pages for hosting  

## Why This Format Was Chosen
A website allows information to be presented in an organized, interactive, and accessible way. This format is effective for college students who are familiar with digital platforms and allows them to learn at their own pace.
